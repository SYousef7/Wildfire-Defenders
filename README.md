Files

index.html – main webpage

sketch.js – game logic + Micro:bit input handling

Requirements

      -You must have Flask installed (pip install flask)   
      -A Micro:bit connected via USB

How to Launch

    -Plug in your Micro:bit

    -Run app.py (this starts the Flask server)

    -Open the local address the terminal gives you

    -Connect the Micro:bit from the webpage when prompted

    -Interact with the prototype
